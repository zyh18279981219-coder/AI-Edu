# AI-Education 项目启动与修改说明

本文档面向项目开发者，说明当前项目在本地如何启动、如何修改、前后端如何联调，以及改动时常用文件在哪里。

## 1. 项目目录

- 项目根目录：`D:\develop\pythonFile\AI-Education2`
- 后端入口：`main.py`
- 后端主程序：`backend/app.py`
- 前端目录：`frontend-vue`
- 前端页面目录：`frontend-vue/src/views`
- 前端接口目录：`frontend-vue/src/api`
- 本地数据库：`data/app.db`
- 文档目录：`docs`

## 2. 本地运行环境

- Python 3.10 及以上
- Node.js 18 及以上
- npm 9 及以上
- Windows + PowerShell

## 3. 安装依赖

### 后端依赖

```powershell
cd D:\develop\pythonFile\AI-Education2
pip install -r requirements.txt
```

### 前端依赖

```powershell
cd D:\develop\pythonFile\AI-Education2\frontend-vue
npm install
```

## 4. 环境变量说明

项目使用根目录下的 `.env` 文件。

当前关键配置包括：

- `model_name`
- `base_url`
- `api_key`
- `embedding_model`

如果你修改了模型、OCR、行业情报分析或 RAG 相关逻辑，先确认 `.env` 可用，再启动项目。

## 5. 如何启动项目

### 启动后端

```powershell
cd D:\develop\pythonFile\AI-Education2
python main.py
```

或使用热重载：

```powershell
cd D:\develop\pythonFile\AI-Education2
uvicorn backend.app:app --host 0.0.0.0 --port 8000 --reload
```

后端启动后访问：

- `http://localhost:8000/`

### 启动前端开发服务器

```powershell
cd D:\develop\pythonFile\AI-Education2\frontend-vue
npm run dev
```

前端开发地址通常为：

- `http://localhost:5173/`

说明：

- 前端通过 Vite 代理访问后端 `/api`
- 如果后端没启动，前端页面会出现接口连接失败

## 6. 如何修改项目

### 改前端页面

主要看这里：

- `frontend-vue/src/views/student`
- `frontend-vue/src/views/teacher`
- `frontend-vue/src/views/admin`
- `frontend-vue/src/components`
- `frontend-vue/src/styles`

常见修改类型：

- 页面布局
- 按钮和表单样式
- 图表展示
- 页面交互逻辑

### 改前端接口调用

主要看：

- `frontend-vue/src/api/client.ts`
- `frontend-vue/src/api/studentTwin.ts`

这里负责：

- axios 配置
- 登录接口
- 学习计划接口
- 学生孪生接口
- 行业情报接口
- 教师与管理端接口

### 改后端接口

主要看：

- `backend/app.py`
- `DigitalTwinModule/digital_twin_api.py`
- `DashboardModule/dashboard_api.py`
- `IndustryIntelligenceModule/api.py`

这些文件负责：

- 登录与会话
- 课程学习接口
- AI 助教、总结、测验
- 学习计划
- 学生数字孪生
- 教师端班级分析
- 行业情报

### 改核心业务逻辑

按模块看：

- 学生数字孪生：`DigitalTwinModule`
- 个性化学习路径：`PathPlannerModule`
- 学习计划：`LearningPlanModule`
- 测验：`QuizModule`
- 总结：`SummaryModule`
- AI 助教：`AgentModule`
- 行业情报：`IndustryIntelligenceModule`
- 数据库：`DatabaseModule`
- 工具层：`tools`

## 7. 前后端联调方式

推荐开发流程：

1. 启动后端 `python main.py`
2. 启动前端 `npm run dev`
3. 前端页面改动后看浏览器热更新
4. 后端逻辑改动后看终端日志

如果改的是接口字段：

1. 先改后端返回结构
2. 再改 `frontend-vue/src/api/studentTwin.ts`
3. 最后改页面展示逻辑
