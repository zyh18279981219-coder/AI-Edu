# 项目启动说明（实体优先版）

## 1. 路径
- 根目录：`D:\develop\pythonFile\AI-Education2`
- 后端：`main.py` / `backend/app.py`
- 前端：`frontend-vue`
- 数据库：`data/app.db`

## 2. 启动

### 后端
```powershell
cd D:\develop\pythonFile\AI-Education2
python main.py
```
或
```powershell
uvicorn backend.app:app --host 0.0.0.0 --port 8000 --reload
```

### 前端
```powershell
cd D:\develop\pythonFile\AI-Education2\frontend-vue
npm install
npm run dev
```

## 3. 当前关键规则

1. 登录使用 `login_id/user_id`。
2. 课程/资料/测试已实体化，接口读写优先 SQLite 实体表。
3. 运行时不再以旧 JSON 作为主写路径。

## 4. 一键自测

```powershell
cd D:\develop\pythonFile\AI-Education2
python scripts\backend_api_selftest.py --base-url http://127.0.0.1:8010
```

## 5. 常见问题

### 5173 白屏
1. 先确认后端可访问：`http://localhost:8000/openapi.json`
2. 前端依赖是否完整：`npm install`
3. 构建检查：`npm run build`

### 改动未生效
- 可能是旧后端进程占用了端口。
- 建议切到新端口（如 `8010`）启动干净进程再验证。
