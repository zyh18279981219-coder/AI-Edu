# 外部数据依赖与对接清单（实体优先）

> 同步日期：2026-04-14

## 1. 学生孪生依赖

| 依赖项 | 当前来源 | 用途 |
|---|---|---|
| 学习进度 | 课程图谱节点完成状态 | 计算 progress 与学习稳定性 |
| 测验成绩 | `quiz_attempts` + quiz 回写 | 计算 quiz_score 与风险判断 |
| 学习时长 | `sessions` | 计算投入度 |
| LLM 互动 | `llm_logs` | 计算互动强度 |
| 节点结构 | `course_nodes` | 节点映射与课程归属 |

## 2. 课程实体依赖

| 依赖项 | 当前来源 | 用途 |
|---|---|---|
| 课程结构 | `courses.payload_json` | 对外图谱返回 |
| 节点列表 | `course_nodes` | 学习节点查询 |
| 资源列表 | `resources` | 节点资源查询与管理 |

## 3. 测试实体依赖

| 依赖项 | 当前来源 | 用途 |
|---|---|---|
| 题目生成 | Quiz 模块 + RAG | 生成测验 |
| 作答记录 | 前端提交 | 形成尝试记录 |
| 结果落库 | `quiz_attempts` | 测试轨迹分析 |
| 孪生回写 | DataCollector | 同步学生孪生节点分 |

## 4. 教师维度依赖

| 依赖项 | 当前来源 | 用途 |
|---|---|---|
| 师生关系 | `teacher_student_links` | 班级范围界定 |
| 学生孪生摘要 | `twin_profiles + twin_profile_nodes` | 教师画像分析 |
| 外部指标 | `/external-sync` 写入 | 教师扩展维度补充 |

## 5. 行业情报依赖

| 依赖项 | 来源 | 用途 |
|---|---|---|
| 招聘数据 | JobSpy / 外部站点 | 岗位抓取 |
| 模型推理 | LLM | 技能抽取与相关性计算 |
| 翻译扩词 | LLM/规则 | 海外岗位检索增强 |

## 6. 对接规范建议

每个新数据源必须提供：
1. 字段清单
2. 更新频率
3. 来源模块
4. 负责人
5. 对接方式（函数/HTTP/文件/消息）

## 7. 排错抓手

- 数据来源审计日志：`data/Log/data_source_audit.log`
- 重点事件：`users_hit`、`twin_profile_hit`、`learning_plans_hit`、`llm_logs_hit`
