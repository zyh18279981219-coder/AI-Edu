# 系统接口说明（实体优先版）

> 同步日期：2026-04-14
> 当前后端实现：课程/资料/测试已实体化，接口读写优先走 SQLite 实体表。

## 1. 认证与会话

### POST `/api/auth/login`
- 说明：统一 JSON 登录。
- 账号标识：`login_id` 或 `user_id`（字符串）。

### POST `/api/logout`
- 说明：退出登录并清理会话。

### GET `/api/current-user`
- 说明：返回当前用户（含 `user_id`、`login_id`）。

## 2. 课程与资源（实体优先）

### GET `/api/knowledge-graph`
- 说明：获取课程图谱。
- 当前读取：优先 `courses.payload_json`。

### GET `/api/learning-nodes`
- 说明：获取学习节点列表。
- 当前读取：优先 `course_nodes`。

### POST `/api/node/resources`
- 说明：获取节点资源。
- 当前读取：优先 `resources`。

### POST `/api/upload`
- 说明：上传节点资源。
- 当前写入：实体主写（同步到 `courses/course_nodes/resources`），并触发 RAG 入库。

### POST `/api/delete-resource`
- 说明：删除节点资源。
- 当前写入：实体主写（同步 `resources` 与课程 payload）。

### POST `/api/pdf/select`
- 说明：设置当前会话 PDF。

### GET `/api/pdf/{path}`
- 说明：读取 PDF 文件。

## 3. 学习与测验

### GET `/api/learning-progress`
- 说明：学习进度统计。
- 当前读取：基于实体课程 payload 计算。

### POST `/api/quiz/start`
- 说明：开始测验。

### POST `/api/quiz/answer`
- 说明：提交测验答案。

### POST `/api/quiz/complete`
- 说明：测验完成并更新节点完成状态。
- 当前写入：
  - 写 `quiz_attempts`（测试实体记录）
  - 同步更新课程实体 payload
  - 同步数字孪生测验分

### POST `/api/quiz/summary`
- 说明：生成测验总结。

## 4. 数字孪生

接口前缀：`/api/digital-twin`

### POST `/api/digital-twin/student-course-profile`
- 输入：`student_id + course_id`
- `student_id` 支持：`login_id` 或 `user_id`
- 输出：课程匹配节点、弱项、趋势、风险、摘要。

### 其他
- `collect/{username}`、`profile/{username}`、`student-profile/{username}`
- `quiz-score`
- `path/generate/{username}`、`path/{username}/current`、`path/{username}/node/{node_id}`
- `teacher-profile/{teacher_username}`、`teacher-profile/{teacher_username}/external-sync`
