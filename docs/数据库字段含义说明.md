# 数据库字段含义说明（data/app.db）

## 1. users（用户主数据）

| 字段 | 类型 | 约束/索引 | 含义 |
|---|---|---|---|
| user_id | INTEGER | PRIMARY KEY，唯一索引 `idx_users_user_id` | 用户唯一数字 ID（物理主键 + 主查询键） |
| login_id | TEXT | UNIQUE 索引 `idx_users_login_id` | 登录账号（主登录键） |
| user_type | TEXT | NOT NULL，索引 `idx_users_type` | 用户类型：`student/teacher/admin` |
| username | TEXT | NOT NULL，唯一约束 `UNIQUE(user_type, username)` | 历史用户名（兼容字段，按类型唯一） |
| password | TEXT |  | 登录密码 |
| display_name | TEXT |  | 展示名称 |
| teacher | TEXT |  | 学生关联教师（历史业务字段） |
| email | TEXT |  | 邮箱 |
| payload_json | TEXT | NOT NULL | 用户完整 JSON 快照 |
| updated_at | TEXT | NOT NULL | 记录更新时间（ISO 字符串） |
索引：`idx_users_type`、`idx_users_user_id`、`idx_users_login_id`

## 2. teacher_student_links（师生关系）

| 字段 | 类型 | 约束/索引 | 含义 |
|---|---|---|---|
| teacher_username | TEXT | NOT NULL，主键组成 | 教师用户名（历史键） |
| student_username | TEXT | NOT NULL，主键组成 | 学生用户名（历史键） |
| updated_at | TEXT | NOT NULL | 关系更新时间 |
| teacher_user_id | INTEGER | 索引 `idx_links_teacher_user_id` | 教师 user_id |
| student_user_id | INTEGER | 索引 `idx_links_student_user_id` | 学生 user_id |

主键：`PRIMARY KEY (teacher_username, student_username)`

## 3. twin_profiles（学生孪生主表）

> 当前已移除 `payload_json`，主表只保留聚合字段。

| 字段 | 类型 | 约束/索引 | 含义 |
|---|---|---|---|
| username | TEXT | PRIMARY KEY | 学生用户名（历史主键） |
| user_id | INTEGER | 索引 `idx_twin_profiles_user_id` | 学生 user_id |
| last_updated | TEXT |  | 业务更新时间 |
| overall_mastery | REAL | DEFAULT 0 | 整体掌握度 |
| updated_at | TEXT | NOT NULL | 记录更新时间 |

## 4. twin_profile_nodes（学生孪生节点明细）

| 字段 | 类型 | 约束/索引 | 含义 |
|---|---|---|---|
| username | TEXT | NOT NULL，主键组成 | 学生用户名 |
| user_id | INTEGER | 索引 `idx_twin_nodes_user_id` | 学生 user_id |
| node_id | TEXT | NOT NULL，主键组成 | 课程节点 ID |
| node_path_json | TEXT |  | 节点路径 JSON 数组 |
| quiz_score | REAL |  | 测验得分 |
| progress | REAL | DEFAULT 0 | 学习进度 |
| study_duration_minutes | REAL | DEFAULT 0 | 学习时长（分钟） |
| llm_interaction_count | INTEGER | DEFAULT 0 | AI 互动次数 |
| mastery_score | REAL | DEFAULT 0 | 节点掌握度 |
| updated_at | TEXT | NOT NULL | 记录更新时间 |

主键：`PRIMARY KEY (username, node_id)`

## 5. twin_history（孪生趋势历史）

| 字段 | 类型 | 约束/索引 | 含义 |
|---|---|---|---|
| username | TEXT | NOT NULL，主键组成 | 学生用户名 |
| snapshot_date | TEXT | NOT NULL，主键组成 | 快照日期（`YYYY-MM-DD`） |
| overall_mastery | REAL | DEFAULT 0 | 当日整体掌握度 |
| payload_json | TEXT | NOT NULL | 历史点 JSON |
| updated_at | TEXT | NOT NULL | 记录更新时间 |
| user_id | INTEGER | 索引 `idx_twin_history_user_id` | 学生 user_id |

## 6. sessions（登录会话）

| 字段 | 类型 | 约束/索引 | 含义 |
|---|---|---|---|
| session_id | TEXT | PRIMARY KEY | 会话 ID（Cookie） |
| username | TEXT | NOT NULL，索引 `idx_sessions_username` | 用户名 |
| user_type | TEXT | NOT NULL | 用户类型 |
| created_at | TEXT |  | 会话创建时间 |
| last_accessed | TEXT |  | 最后访问时间 |
| current_pdf_path | TEXT |  | 当前学习 PDF |
| current_node | TEXT |  | 当前学习节点 |
| payload_json | TEXT | NOT NULL | 会话完整 JSON |
| updated_at | TEXT | NOT NULL | 记录更新时间 |
| user_id | INTEGER | 索引 `idx_sessions_user_id` | 用户 user_id |

## 7. user_states（用户扩展状态）

| 字段 | 类型 | 约束/索引 | 含义 |
|---|---|---|---|
| username | TEXT | PRIMARY KEY | 用户名 |
| payload_json | TEXT | NOT NULL | 扩展状态 JSON |
| updated_at | TEXT | NOT NULL | 记录更新时间 |
| user_id | INTEGER | 索引 `idx_user_states_user_id` | 用户 user_id |

## 8. llm_logs（LLM 调用日志）

| 字段 | 类型 | 约束/索引 | 含义 |
|---|---|---|---|
| id | INTEGER | PRIMARY KEY AUTOINCREMENT | 日志主键 |
| timestamp | TEXT | 索引 `idx_logs_timestamp` | 调用时间 |
| username | TEXT | 索引 `idx_logs_username` | 用户名 |
| module | TEXT |  | 调用模块 |
| model | TEXT |  | 模型名 |
| payload_json | TEXT | NOT NULL | 请求/响应完整 JSON |
| user_id | INTEGER | 索引 `idx_llm_logs_user_id` | 用户 user_id |

## 9. learning_plans（学习计划）

| 字段 | 类型 | 约束/索引 | 含义 |
|---|---|---|---|
| username | TEXT | NOT NULL，主键组成，索引 `idx_plans_username` | 用户名 |
| filename | TEXT | NOT NULL，主键组成 | 计划文件名 |
| plan_path | TEXT |  | 文件路径 |
| category | TEXT |  | 分类（如 `global/user/path`） |
| payload_json | TEXT | NOT NULL | 计划内容 JSON |
| updated_at | TEXT | NOT NULL | 记录更新时间 |
| user_id | INTEGER | 索引 `idx_learning_plans_user_id` | 用户 user_id |

## 10. courses（课程主数据）

| 字段 | 类型 | 约束/索引 | 含义 |
|---|---|---|---|
| course_id | TEXT | PRIMARY KEY | 课程唯一 ID |
| course_name | TEXT |  | 课程名称 |
| source_path | TEXT |  | 课程来源文件路径 |
| payload_json | TEXT | NOT NULL | 课程原始结构快照 JSON |
| updated_at | TEXT | NOT NULL | 记录更新时间 |

## 11. course_nodes（课程节点）

| 字段 | 类型 | 约束/索引 | 含义 |
|---|---|---|---|
| course_id | TEXT | NOT NULL，主键组成，索引 `idx_course_nodes_course` | 课程 ID |
| node_id | TEXT | NOT NULL，主键组成 | 节点唯一 ID（在课程内唯一） |
| node_name | TEXT |  | 节点名称 |
| node_path_json | TEXT |  | 节点路径 JSON |
| depth | INTEGER | DEFAULT 0 | 节点深度 |
| parent_node_id | TEXT |  | 父节点 ID |
| payload_json | TEXT | NOT NULL | 节点原始数据 JSON |
| updated_at | TEXT | NOT NULL | 记录更新时间 |

主键：`PRIMARY KEY (course_id, node_id)`

## 12. resources（资料实体）

| 字段 | 类型 | 约束/索引 | 含义 |
|---|---|---|---|
| resource_id | INTEGER | PRIMARY KEY AUTOINCREMENT | 资料唯一 ID |
| course_id | TEXT | NOT NULL，索引 `idx_resources_course_node` | 课程 ID |
| node_id | TEXT | NOT NULL，索引 `idx_resources_course_node` | 节点 ID |
| resource_path | TEXT | NOT NULL，索引 `idx_resources_path` | 资料路径 |
| resource_type | TEXT |  | 资料类型（如 `pdf`） |
| title | TEXT |  | 资料标题（默认文件名） |
| payload_json | TEXT | NOT NULL | 资料扩展信息 JSON |
| created_at | TEXT | NOT NULL | 创建时间 |
| updated_at | TEXT | NOT NULL | 更新时间 |

唯一约束：`UNIQUE (course_id, node_id, resource_path)`

## 13. quiz_attempts（测试实体）

| 字段 | 类型 | 约束/索引 | 含义 |
|---|---|---|---|
| attempt_id | INTEGER | PRIMARY KEY AUTOINCREMENT | 测试记录唯一 ID |
| user_id | INTEGER | 索引 `idx_quiz_attempts_user_id` | 用户 ID |
| username | TEXT | 索引 `idx_quiz_attempts_username` | 用户名（兼容） |
| course_id | TEXT | 索引 `idx_quiz_attempts_course_id` | 课程 ID |
| node_id | TEXT |  | 节点 ID |
| score | REAL |  | 得分 |
| total | REAL |  | 总分 |
| passed | INTEGER | DEFAULT 0 | 是否通过（0/1） |
| payload_json | TEXT | NOT NULL | 记录快照 JSON |
| created_at | TEXT | NOT NULL，索引 `idx_quiz_attempts_created_at` | 提交时间 |
