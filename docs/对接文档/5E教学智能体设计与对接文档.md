# 5E教学智能体设计与对接文档

# 一、有哪些功能

## 提供学生与智能体交互的入口。根据学生、课程数据及学生的回复动态调用智能体，带领学生学习课程。 

# 二、数据库表设计（已有）

由Agent框架提供，不与其他数据冲突，也不需要直接读取该数据库 

# 三、需要别人提供的数据
需要学生数字孪生提供接口： 

传入学生id和课程id，返回该学生在该课程的学习状态明细 

需要课程数字孪生提供接口： 

传入课程id，返回课程名称 

传入课程id，返回该课程的所有相关资料（名称、链接或其他访问方式） 

传入课程id，返回该课程的相关测试卷或作业 

传入课程id和学生id，返回学生该课程的所有测试卷id 

传入学生id和测试卷id，返回学生该测试的所有作答信息和成绩
